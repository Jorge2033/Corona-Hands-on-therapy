# Corona Hands-On Therapy — Landing Page

Welcome to the official repository for the **Corona Hands-On Therapy** landing page, a physical therapy, chiropractic care, and acupuncture clinic located in Elmhurst, NY. 

This website has been built following modern web development standards to provide an optimal, fast, and responsive user experience for patients seeking recovery-focused care.

## 🛠️ Tech Stack & Features

- **Next.js 14 (App Router):** Leveraging React's latest features and built-in API routes for serverless form processing.
- **TypeScript:** Ensuring robust data typing and eliminating runtime errors on patient contact forms.
- **CSS Modules:** Scalable, component-scoped styling without bloating the application with external UI libraries.
- **Nodemailer:** Integrated secure communication system for seamless email appointment requests.

## 👥 Repository Architecture

This repository uses a structured workflow to maintain stability:
- **`main` branch:** Contains production-ready documentation and structural overviews. (Protected branch).
- **`dev` branch:** Contains the active application source code, local setup configuration guides, environment variable blueprints, and deployment pipeline workflow details.

For security reasons, proprietary source code and configuration parameters are maintained in dedicated development streams.
